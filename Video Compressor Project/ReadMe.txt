Manual:

* Without GUI:

1 . for Compression Run Mars function as
	
	ResultFilePath = Mars('video.mpg');
		% full path if video is present
		% outside the current dir.

2 . Note down the Compression Percentage

	
3 . for Decompression Run DeMars function as

	VideoObj = DeMars('video_mpg_mars.mat');
		% input the path of the intermediate file
		% formate should be in ' .mat ' extension

4 . Result Video path is displayed and video is played.
